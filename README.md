# Squash Academy — Simple Static Site

This is a minimal static website for a squash academy (HTML + CSS).

Files
- `index.html` — main page
- `styles.css` — styles

Preview locally
1. Open `index.html` in your browser directly, or run a simple web server from this folder:

```bash
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

Feel free to tell me if you want additional pages, images, or a contact backend.
